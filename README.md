# DMMサービス紹介・おすすめ情報まとめ

これは、DMM.comおよびその関連サービスで提供されるエンターテインメント関連のサービスや多様なコンテンツに関する紹介記事を掲載するために作成された、個人運営のウェブサイトです。

## ✨ 特徴

- **Three.jsによる動的な背景**: サイトの背景には、Three.js、SimplexNoise、Chroma.jsを使用して生成された、美しく動的なチューブアニメーションが描画されています。
- **レスポンシブデザイン**: ウェブサイトは、デスクトップ、タブレット、モバイルデバイスなど、さまざまな画面サイズに対応するレスポンシブデザインを採用しています。
- **統一された美的感覚**: サイト全体のすべてのページは、一貫した視覚的スタイルとユーザーエクスペリエンスを提供するために、同じレイアウトとスタイリングを共有しています。
- **軽量なセットアップ**: このプロジェクトは、ビルドプロセスや複雑な依存関係を必要としない、シンプルなHTML、CSS、JavaScriptファイルで構成されています。

## 📄 ページ

このプロジェクトには、以下のHTMLページが含まれています：

- `index.html`: ウェブサイトのメインランディングページ。
- `about.html`: サイトとその目的についての情報。
- `contact.html`: 問い合わせ先情報。
- `privacy.html`: プライバシーポリシーの詳細。

## 🛠️ 使用技術

このプロジェクトでは、以下の技術とライブラリが使用されています：

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
- **[Three.js](https://threejs.org/)**: 3Dグラフィックス用
- **[SimplexNoise.js](https://github.com/jwagner/simplex-noise.js)**: ノイズ生成用
- **[Chroma.js](https://gka.github.io/chroma.js/)**: カラー操作用

## 🚀 使い方

このウェブサイトを閲覧するには、お好みのウェブブラウザで`index.html`ファイルを開くだけです。

```bash
# 例
open index.html
```
